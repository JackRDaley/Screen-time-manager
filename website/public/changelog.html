Changelog

Actively improving the focus experience.

Recent updates make Saturn easier to start, clearer to understand, and more useful during repeat focus sessions.

Adaptive popup dashboard refresh

Redesigned the extension popup with a sharper Saturn interface, responsive Today summaries, clearer journey and profile stats, live ranking actions, and more consistent typography across every tab.

More precise analytics signals

Separated personal insight impressions from interactions and added preset application tracking so product analytics better reflect what users actually see and use.

Quick-start presets

Added Study Mode, Deep Work Mode, and Sleep Mode so new users can create useful limits without manually building every rule from scratch.

Clearer blocked-page tracking

Added better tracking for blocked page views and blocked-page actions so users can see when distracting sites are being reopened out of habit.

Activity dashboard improvements

Refined dashboard stats around screen time, visits, blocked attempts, and time saved so daily behavior is easier to scan.

Premium handoff improvements

Improved the checkout handoff flow so paid activation can return to the correct extension install.

Saturn
